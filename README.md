# 1. CMSF(COMMON-MYSQL-STORED-FUNCTION)とは

 CMSF - COMMON MYSQL STORED FUNCTION(PROCEDURE)は
 MYSQLを使いやすく便利にするための汎用的な共通関数群の総称です。
あれば便利だけどまだない関数を実装し、全国津々浦々、
様々なプロジェクトで共用できる汎用的なプログラムツールの作成を目指しています。
